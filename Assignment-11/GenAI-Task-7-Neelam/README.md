Book Format Market Share Visualization

This task involves analyzing and visualizing book sales data using Pandas and Matplotlib. It plots a pie chart to display the market share and proportion of different book formats (e.g., paperback, hardcover) in the dataset.

Prerequisites:
Ensure you have the required packages installed. These are listed in pyproject.toml:
pandas
matplotlib

If you are using uv, you can run the script using the workspace environment.

How to Run

Open your terminal at the workspace root directory.
Navigate to the Assignment-11 directory (so that the script can find book_sales.csv in its working directory):
cd Assignment-11
Run the script using the project's virtual environment:
.venv\Scripts\python.exe GenAI-Task-7-Neelam\main.py
Or if you have uv installed:
uv run GenAI-Task-7-Neelam\main.py
